O projeto Ugla foi desenvolvido com o objetivo de minimizar esforços no desenvolvimento FrontEnd de sistemas, trazendo consistencia e segurança para projetos que necessitam conversar entre si ou, que sejam totalmente independentes, mantendo uma identidade única e de fácil entendimento.

Todos os componentes desenvolvidos possuem sua base construtiva em 3 grandes pilares:

Acessibilidade

Queremos que abranger o maior número de usuários possível, sem limitar ninguém. Por isso cada componente possuem em seu desenvolvimento a preocupação com a implementação correta da semântica e de recursos de acessibilidade.

Consistência:

O sistema precisa ser simples, objetivo e claro. Por isso usamos um padrão de ícones e formatos de texto, assim como componentes bem especificados para que não gere confusão nem desentendimento por parte do usuário. Queremos simplificar e agilizar.

Independente:

Apesar de construido como uma biblioteca Angular, procuramos usar o máximo de recursos próprios, não misturando tecnologias que não são de fato cruciais. Assim, esse é um projeto que tenta ao máximo utilizar 100% dos recursos oferecidos pelo Angular.

O Ugla oferece diversos componentes, diretivas e serviços para o desenvolvimento de sistemas. No menu superior criamos sessões para melhor organizar e categorizar cada elemento disponibilizado.

Temas

O Ugla trabalha com uma paleta de temas, o que premite mudar facilmente a cor base do sistema. Essa cor é definida no momento em que importa o Ugla para o seu projeto com o comando a seguir: UglaModule.forRoot({themeName: Themes.YELLOW})

Inslando e configurando

A instalação é um processo bem simples, conheça mais detalhes no Github do projeto.

Instação no package.json

npm i ugla

Configurando

No app.module.js importe o u UglaModule da seguinte maneira:

UglaModule.forRoot({themeName: Themes.YELLOW})

As opções de temas são: WHITE, AQUAMARINE, RED, GREEN, YELLOW, GRAY e PURPLE
